Whook content slider is a Post, Page and Specific Category's Posts Slider

=== Description ===

Wordress website or developer need to showcase work or case study on home page of website. Bloggers need to showcase their featured blogs on the home page, or Internal pages of website.

Whook content slider comes up with feature to show case your premium posts or pages or categories on the showcase. You can show the excerpt or featured image in the widget. Also featured image will be focused as parallax effect which gives your website a different style from a usual kind of website.

It’s a combination of parallax and carousal combination.

=== Shortcode to use ===
1. <strong>To load particular pages in the carousal.</strong> `[dt_slider title="Related project" page_id="847,892" order_by="ID" order="asc"]`
1. <strong>To load posts from a particular category</strong> `[dt_slider title="Related project" category_id="10" order_by="ID" order="asc"]`
1. <strong>To load posts, belong to different category or independent posts.</strong> `[dt_slider title="Related project" post_id="45,52" order_by="ID" order="asc"]`

= Parameters to be used in shortcode =
* <strong>Title:</strong> Title of the plugin or widget needs to be get displayed.
* <strong>Page_id:</strong> If you want to load page ids.
* <strong>Category_id:</strong> If you want to load posts from particular category.
* <strong>Post_id:</strong> If you want to load particular posts on the layout.
* <strong>Order_by:</strong> On which order you want to fetch details. `["ID","post_date","post_title"]`
* <strong>Order:</strong> In which order you want to load posts/pages. `["asc","desc"]`


=== Installation ===
* Download plugin from WordPress Plugin directory.
* Upload it to the plugins folder, unzip it there.
* Activate the plugin from admin plugins dashboard section.
